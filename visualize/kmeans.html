📍 What is K-Means Clustering?

K-Means is an unsupervised machine learning algorithm used for clustering. It groups data into K distinct, non-overlapping clusters based on similarity. It is widely used in pattern recognition, image compression, and customer segmentation.

🔧 How Does K-Means Work?

  1. Choose the number of clusters K
  2. Initialize K centroids randomly
  3. Assign each data point to the nearest centroid
  4. Recalculate centroids by averaging all points in each cluster
  5. Repeat steps 3–4 until centroids no longer change significantly

🧪 Example Use Cases

  • Customer segmentation in marketing
  • Market basket analysis
  • Document or text clustering
  • Image compression
  • Grouping search results

📊 Equation Used

The objective of K-Means is to minimize the Within-Cluster Sum of Squares (WCSS): 


    J = Σ Σ ||xi - μj||²
        j   i

Where xi is a data point, and μj is the centroid of cluster j.

✅ Advantages

  • Simple and fast for large datasets
  • Works well when clusters are well-separated
  • Easy to implement and interpret

⚠️ Limitations

  • Must specify the number of clusters K in advance
  • Not suitable for non-spherical or overlapping clusters
  • Sensitive to outliers and initial centroid placement
