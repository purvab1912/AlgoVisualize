K-Means is an unsupervised machine learning algorithm used for clustering. It groups data into K distinct, non-overlapping clusters based on similarity. It is widely used in pattern recognition, image compression, and customer segmentation.

  1. Choose the number of clusters K
  2. Initialize K centroids randomly
  3. Assign each data point to the nearest centroid
  4. Recalculate centroids by averaging all points in each cluster
  5. Repeat steps 3â€“4 until centroids no longer change significantly
