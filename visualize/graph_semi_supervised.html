üåê What is Graph-Based Semi-Supervised Learning?

Graph-Based Semi-Supervised Learning is a class of algorithms that use graph structures to represent the relationships or similarities between data points, both labeled and unlabeled. The idea is to propagate label information smoothly over the graph, assuming that connected or similar nodes are likely to share the same label.

üìö Detailed Concept

In graph-based methods, data points are represented as nodes in a graph, and edges indicate similarity or affinity between points (often weighted by a similarity function). The label information is propagated from labeled nodes to unlabeled ones through edges, often by minimizing an energy function that enforces label smoothness on the graph. This leverages the geometric structure of data and the cluster assumption: points in the same cluster tend to have the same label.

üõ†Ô∏è Types of Graph-Based Semi-Supervised Learning

  • Label Propagation: Labels are spread iteratively from labeled nodes to unlabeled nodes based on graph connectivity until convergence.
  • Label Spreading: Similar to label propagation but with normalization to ensure stability and better handling of noisy labels.
  • Graph Regularization Methods: Combine supervised loss with a smoothness term on the graph Laplacian to encourage similar nodes to have similar labels.
  • Manifold Regularization: Extends graph regularization by incorporating manifold assumptions in the learning process, improving generalization.
  • Graph Neural Networks (GNNs): Use neural networks to learn node embeddings that capture graph structure and labels, enabling semi-supervised learning in complex graphs.

üöÄ Applications

  • Social network analysis for community detection and influence prediction.
  • Image and video segmentation where pixels or regions are nodes in a graph.
  • Recommendation systems using user-item interaction graphs.
  • Bioinformatics for predicting protein functions or gene interactions.
