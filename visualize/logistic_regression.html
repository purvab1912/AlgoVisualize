üìå What is Logistic Regression?

Logistic Regression is a supervised machine learning algorithm used for classification tasks. It predicts the probability that a given input belongs to a particular category, usually between 0 and 1.

It is particularly used when the target variable is binary (e.g., Yes/No, 0/1, True/False).

‚úÖ Common Use Cases

  • Email spam detection
  • Credit card fraud detection
  • Customer churn prediction
  • Medical diagnosis (e.g., disease presence or absence)
  • Loan approval predictions

üî¢ Types of Logistic Regression

  • Binary Logistic Regression: 2 possible outcomes (e.g., spam or not spam)
  • Multiclass Logistic Regression: More than 2 categories (e.g., type of animal: cat, dog, rabbit)
  • Multilabel Classification: Each input can belong to multiple classes (e.g., text tagging)

üìù Real-World Example

A bank wants to predict whether a customer will default on a loan. Based on past data:

  • Input (X): Income, Credit Score, Debt
  • Output (Y): Default = 1, No Default = 0

The logistic regression model outputs a probability (e.g., 0.82). If it's over 0.5, the system flags the customer as high-risk (default = 1).
