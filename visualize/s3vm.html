Semi-Supervised Support Vector Machine (S3VM) is an extension of the classic Support Vector Machine (SVM) that leverages both labeled and unlabeled data to improve classification performance. While traditional SVMs are supervised and use only labeled data, S3VMs utilize the unlabeled data to find a decision boundary that not only separates the labeled samples well but also avoids cutting through dense regions of the data distribution.

üìö Detailed Concept

S3VM tries to find a decision boundary with a large margin (like regular SVM) but also incorporates the unlabeled data by encouraging the boundary to lie in low-density regions, reflecting the cluster assumption: data points in the same cluster are likely to share the same label. This makes the classifier more robust and accurate when labeled data is limited but unlabeled data is plentiful.

üõ†Ô∏è Types of Semi-Supervised SVM

  • Transductive SVM (TSVM): Directly finds the decision boundary that maximizes margin on labeled and unlabeled data simultaneously. It is a common approach but computationally expensive.
  • Graph-based SVM: Combines SVM with graph-based regularization to incorporate similarity between samples.
  • Entropy Regularized SVM: Adds an entropy term to encourage confident predictions on unlabeled data.
  • Co-Training SVM: Uses multiple SVMs trained on different feature views, applying co-training principles to leverage unlabeled data.

üöÄ Applications

  • Text classification where only some documents are labeled.
  • Image recognition with scarce labeled images.
  • Bioinformatics for gene or protein function prediction.
  • Speech and audio classification with limited annotations.
