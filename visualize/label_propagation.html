Label Propagation is a graph-based semi-supervised learning algorithm that spreads labels from a small set of labeled data points to a large set of unlabeled data points using the structure of the data. It works by treating data as nodes in a graph, where edges represent similarities between points. Labels ‚Äúpropagate‚Äù through the graph iteratively until convergence.

Label propagation iteratively updates the labels of unlabeled nodes based on their neighbors‚Äô labels, smoothing the label information across the graph. Eventually, each node‚Äôs label stabilizes, effectively assigning labels to previously unlabeled data.
